利用空间量算采集点位

2019-09-10 3 min read # 前端开发笔记

在高亮建筑物模型的一层或者房间时，为了便于后续进行属性查询或其他操作，需要先采集点位制作头文件以方便调用读取。

在double click时，添加如下代码:

for(var i = 0; i <=positions.length; ++i){
                let cartographic = Cesium.Cartographic.fromCartesian(positions[i]);
                var lon = Cesium.Math.toDegrees(cartographic.longitude).toFixed(6);
                var lat = Cesium.Math.toDegrees(cartographic.latitude).toFixed(6);
                // var hei = cartographic.height;
                // console.log('position'+i+':'+lon+','+lat);
                console.log(lon+','+lat+',');
            }
